# Minha Vida Financeira

Sistema de controle financeiro pessoal com painel responsivo e importação de extratos bancários.

## Recursos disponíveis

- Dashboard financeiro responsivo
- Resumo de receitas, despesas, saldo e metas
- Importação de extratos em CSV e OFX
- Identificação automática de receitas e despesas
- Sugestão de categorias
- Revisão dos lançamentos antes da confirmação
- Layout adaptado para computador e celular
